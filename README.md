# MERN Full Stack Application

A modern **full-stack web application** built using the **MERN stack (MongoDB, Express.js, React, and Node.js)**. This project demonstrates how a complete production-style application is structured, including backend APIs, frontend UI, authentication, and database integration. The goal of this project is to showcase practical full-stack development using industry best practices and scalable architecture.

## Overview

This application follows a **client–server architecture** where the frontend is developed using **React** and the backend is powered by **Node.js and Express.js**. MongoDB is used as the database to store and manage application data. The system communicates through **RESTful APIs**, enabling smooth interaction between the frontend and backend.

The project focuses on real-world development patterns such as modular folder structure, API routing, secure authentication, and proper state management on the frontend.

## Features

* Full stack MERN architecture
* RESTful API development with Express.js
* MongoDB database integration using Mongoose
* User authentication and authorization
* Complete CRUD operations (Create, Read, Update, Delete)
* React-based responsive frontend interface
* API communication using Axios or Fetch
* Scalable and modular backend structure
* Clean and maintainable code organization

## Installation

Clone the repository:

```
git clone https://github.com/ajitdev01/mern-fullstack.git
```

Navigate to the project folder:

```
cd mern-fullstack
```

Install backend dependencies:

```
npm install
```

Install frontend dependencies:

```
cd client
npm install
```

Run the backend server:

```
npm run server
```

Run the frontend application:

```
npm start
```
